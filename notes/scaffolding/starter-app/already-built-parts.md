# Already Built Parts
There are a couple of pieces that we have built before, that should be used in the scaffolding and starter app.


## Docx2Md
This is a custom go package that converts contracts in .docx format into markdown. It uses pandoc and applies lots of custom tweaks to get clause numbering and chunking right.
Lives at: ~/accordli-dev/evolver/docx-files/contracts-enhanced/tools/docx2md-go
To Use: Copy the whole package into the source for the app.
